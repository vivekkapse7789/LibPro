# LibPro: Cinematic Library Management System

LibPro is a sophisticated, full-stack library management solution designed with a modern "Cinematic Gallery" aesthetic.

## ✨ Key Features

CRUD Operations: Create, Read, Update, and Delete logic via PHP.

Database Management: Structured Query Language (SQL) for persistent storage.

Real-time Inventory: Visual badges for book availability.

Advanced Styling: Using modern CSS properties to enhance user retention and engagement.



## 🚀 Tech Stack 

Frontend : HTML5, CSS3, JavaScript (ES6+) 

Backend : PHP 

Database : MySQL 

Tools : XAMPP, Git, VS Code
