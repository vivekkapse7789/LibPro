\# LibPro: Cinematic Library Management System



!\[Version](https://img.shields.io/badge/version-Final%20Cinematic%20Edition-blue)

!\[Tech Stack](https://img.shields.io/badge/tech-HTML5%20|%20CSS3%20|%20JS%20|%20PHP%20|%20MySQL-orange)



\*\*LibPro\*\* is a sophisticated, full-stack library management solution designed with a modern "Cinematic Gallery" aesthetic.



\## ✨ Key Features

\* \*\*Cinematic Hero Section:\*\* Animated "Aurora" mesh gradient.

\* \*\*Curated Book Gallery:\*\* Interactive "Light Glow" hover effects.

\* \*\*Real-time Inventory:\*\* Visual badges for book availability.



\## 🚀 Tech Stack

| Component | Technology |

| :--- | :--- |

| \*\*Frontend\*\* | HTML5, CSS3, JavaScript (ES6+) |

| \*\*Backend\*\* | PHP |

| \*\*Database\*\* | MySQL |

| \*\*Tools\*\* | XAMPP, Git, VS Code |
